# Markdown 测试文档

这是一个用于测试 RAG 系统对 Markdown 文件支持能力的文档。

## 1. 介绍
Markdown 是一种轻量级标记语言，创始人为 John Gruber。它允许人们使用易读易写的纯文本格式编写文档，然后将其转换成有效的 XHTML（或者 HTML）。

## 2. 特点
- **易读易写**: 即使不转换成 HTML，Markdown 文件本身也具有很高的可读性。
- **格式丰富**: 支持标题、列表、代码块、引用等。

## 3. 测试目标
确保 RAG 系统能够正确解析此文件的内容，并回答相关问题。

例如：Markdown 的创始人是谁？
